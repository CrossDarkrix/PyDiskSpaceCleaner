# PyDisk Cleaner(iOS only Tested.)
*PyDisk CleanerはPythonベースで作られたDisk Cleanerです。*

# Features
*PyDisk Cleanerは2種類またはその両方を使ったディスククリーナーで、*

*・Ranadm Fill*

*・Zero Fill*

*といった構成で作られており、3つのファイルに徐々にデータを書き込んでいきディスクのクリーニングを実行します。*

## Usage
*`python3 PyDiskCleaner.py`*
